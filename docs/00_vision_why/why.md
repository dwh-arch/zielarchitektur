# Vision (Why)
Wir bauen eine Data Platform, die:
- schnell neue Datenprodukte ermöglicht (Time-to-Value),
- reproduzierbar & testbar ist (CI/CD + Data Quality),
- lokal im HomeLab und/oder in Cloud-Umgebungen betrieben werden kann,
- sowohl klassische BI als auch Data Science/AI unterstützt,
- Integrations- und Automations-Patterns (n8n, Agents, MCP) integriert.
